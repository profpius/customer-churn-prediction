
# Visualizations

This folder contains all charts and plots generated during the Customer Churn Prediction project.

## Contents

| File | Description |
|------|-------------|
| target_distribution.png | Distribution of churn vs non-churn customers |
| model_comparison.png | Model performance comparison across metrics |
| roc_curves.png | ROC curves for all trained models |
| confusion_matrices.png | Confusion matrices for all models |
| feature_importance_rf.png | Random Forest feature importance |
| feature_importance_xgb.png | XGBoost feature importance |
| shap_summary_bar.png | SHAP summary bar plot — global feature impact |
| shap_summary_beeswarm.png | SHAP beeswarm plot — feature impact distribution |
| shap_dependence_wallet.png | SHAP dependence plot for points_in_wallet |

## Notes

- All plots were generated using Matplotlib and Seaborn
- SHAP plots were produced using the SHAP library for model explainability
